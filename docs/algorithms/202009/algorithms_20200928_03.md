## 문제

- BOJ-입출력-10952

---

### 1. 이해

- input:두 정수
- output: 두 정수의 합
- 종료 조건: 두 정수 값이 0이다.

### 2. 계획

- while 반복문 안에서 if문으로 종료 조건 추가

### 3. 실행

```java
package easy;

import java.util.Scanner;

public class Main05 {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);

        while(true){

            int num1 = sc.nextInt();
            int num2 = sc.nextInt();

            if(num1 == 0 && num2 == 0){
                    sc.close();
                    return;
            }

            System.out.println(num1+num2);
        }
    }
}

```
