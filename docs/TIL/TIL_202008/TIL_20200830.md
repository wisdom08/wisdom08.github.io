# Finding

### 스프링 컨텍스트

- 스프링의 컨텍스트 클래스들은 객체를 생성하고 관리하는 컨테이너의 기능을 한다.
- 컨테이너로부터 빈을 찾기 위해서는 컨텍스트가 있어야 한다.

### AOP

- AOP는 문제를 바라보는 관점을 기준으로 프로그래밍 하는 기법
- 가장 중요한 개념은 횡단 관점의 분리이다.

### AOP와 횡단 관점
